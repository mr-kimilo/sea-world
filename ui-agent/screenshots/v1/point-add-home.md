## 交互说明

- 维度卡片点击 → 选中高亮，可多选
- 选中后维度卡片增加选中态（边框高亮 + 勾选标记）
- 自定义维度点击 → 弹出自定义分数输入
- 备注输入框可选填
- "确认加分" → 提交数据 → 成功提示 → 返回首页
- 每个维度可点击修改默认分数

## 配色方案

- 维度卡片: 白色背景 + 浅蓝边框，圆角
- 选中态: 蓝色边框 (#0077B6) + 浅蓝背景 (#E6F2FF)
- 分数文字: 金色/橙色 (#FF8C00)
- Emoji 图标: 大号居中
- 确认按钮: 海洋渐变 (#0077B6 → #00B4D8)

## 海洋主题元素

- 卡片使用气泡般的圆润设计
- 背景有极淡的水波纹
- 选中态使用波浪形边框
