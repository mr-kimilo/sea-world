## 交互说明

- 用户信息卡片点击 → 编辑个人资料
- 菜单项点击 → 跳转对应功能页
- 语言切换 → 弹出语言选择列表
- 退出登录 → 确认弹窗 → 清除登录态 → 跳转登录页
- 底部 TabBar 切换页面

## 配色方案

- 用户卡片: 海洋蓝渐变背景，白色文字
- 功能图标: 蓝色/彩色
- 菜单文字: 深灰 (#333)
- 菜单分隔线: 浅灰 (#E8E8E8)
- 退出按钮: 红色 (#E53E3E)
- 分组标题: 灰色小字

## 海洋主题元素

- 用户卡片使用海洋波浪渐变背景
- 分区使用波浪形分割线
- 图标使用海洋主题色
