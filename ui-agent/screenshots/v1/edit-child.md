## 组件清单

| # | 组件 | 类型 | 位置 | 说明 |
|---|------|------|------|------|
| 1 | 导航栏 | AppBar | 顶部 | 返回 ←、标题"编辑资料" |
| 2 | 当前头像 | Avatar | 顶部 | 圆形头像 + 孩子名 + "点击更换头像" |
| 3 | 昵称输入框 | TextInput | 表单第 1 行 | 已预填当前昵称 |
| 4 | 真实姓名输入框 | TextInput | 表单第 2 行 | 已预填当前姓名 |
| 5 | 出生日期选择器 | DatePicker | 表单第 3 行 | 已预填当前日期 |
| 6 | 性别选择器 | RadioGroup | 表单第 4 行 | 当前性别已选中 |
| 7 | 积分展示 | InfoRow | 表单第 5 行 | 只读显示"当前积分: 150" |
| 8 | 保存按钮 | Button (Primary) | 表单底部 | "保存修改" |
| 9 | 删除按钮 | Button (Danger) | 保存下方 | "删除孩子"，红色 |
| 10 | 积分历史链接 | Link | 底部 | "查看积分历史 →" |

## 交互说明

- 与 add-child 一致，但所有字段预填当前值
- 积分字段只读显示，不可编辑
- 保存 → 校验 → API 更新 → 成功返回
- 删除 → 强确认弹窗 → 二次确认 → 删除成功
- "查看积分历史" → 跳转该孩子的积分历史页
- 头像点击 → 更换照片

## 配色方案

- 与 add-child 保持一致
- 积分文字: 金色 (#FFD700) ⭐ 加粗
- 删除按钮: 红色 (#E53E3E)
- 积分历史链接: 蓝色链接色

## 海洋主题元素

- 头像边框使用波浪或珊瑚装饰
- 积分展示带有气泡装饰背景
- 整体风格与 add-child 统一
