## 交互说明

- 输入框获取焦点时边框高亮（海洋蓝）
- 密码输入框有 👁 切换明文/密文
- 登录按钮 → 校验表单 → 调 API → 成功跳转首页 / 失败显示错误
- 注册链接 → 跳转注册页
- 忘记密码 → 跳转找回密码流程
- 第三方登录 → 调用对应 OAuth 授权

## 配色方案

- 背景: 海洋蓝渐变（从上到下由深蓝到浅蓝）
- Logo/标题: 白色 (#FFFFFF)
- 输入框: 白色背景，圆角，浅灰边框
- 输入框焦点: 蓝色边框 (#0077B6)
- 登录按钮: 渐变蓝色 (#0077B6 → #00B4D8)
- 链接文字: 浅蓝 (#90E0EF)
- 社交图标: 白底彩色

## 海洋主题元素

- 全屏海洋蓝渐变背景
- 波浪动画在底部浮动
- 气泡点缀上升动画
- Logo 鱼形图标
- 按钮使用海洋渐变

## 优化建议

1. 增加背景波浪动画增强沉浸感
2. Logo 可增加微动效（鱼游动）
3. 输入框可增加海洋主题的装饰（圆角更大、阴影更柔和）
4. 登录按钮增加按压交互动效
5. 可增加"自动登录"选项开关
6. 第三方登录图标建议统一风格
