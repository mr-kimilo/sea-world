## 交互说明

- 菜单项点击 → 关闭抽屉并跳转到对应页面
- 当前所在页面菜单项高亮
- 点击遮罩层 → 关闭抽屉
- 从屏幕左侧右滑 → 打开抽屉
- 退出登录 → 弹出确认对话框 → 清除登录状态 → 跳转登录页
- 用户资料区域点击 → 个人中心

## 配色方案

- 抽屉背景: 白色 / 浅灰 (#F8F9FA)
- 菜单文字: 深灰 (#333)
- 选中高亮: 海洋蓝 (#0077B6) + 浅蓝背景
- 退出按钮: 红色 (#E53E3E)
- 遮罩层: 黑色半透明 (rgba(0,0,0,0.5))
- 用户区域: 海洋蓝渐变背景

## 海洋主题元素

- 用户头部区域使用海洋蓝渐变
- 分割线使用波浪形装饰
- 图标风格统一为圆润线条

## 优化建议

1. 用户区域可增加海洋波浪动画背景
2. 菜单项可增加微小的交互动效（右滑指示器）
3. 退出登录建议增加二次确认弹窗（当前可能已有）
4. 菜单项可增加角标（Badge）显示未读/待办数量
5. 底部退出按钮与上方菜单间增加视觉分隔
