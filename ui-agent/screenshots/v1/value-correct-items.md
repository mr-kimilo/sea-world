## 组件清单

| # | 组件 | 类型 | 位置 | 说明 |
|---|------|------|------|------|
| 1 | 导航栏 | AppBar | 顶部 | 返回 ←、标题"价值观纠正" |
| 2 | 引导说明 | GuideText | 导航栏下方 | "选择需要纠正的行为" 提示文字 |
| 3 | 纠正行为卡片 | Card | 列表区域 | 每项：严重程度色条 + 行为名称 + 描述 + 扣分数 + 确认按钮 |
| 4 | 说谎卡片 | Card | 第 1 项 | 🔴 红色标识，"说谎" -5 分 |
| 5 | 打人卡片 | Card | 第 2 项 | 🟡 黄色标识，"打人" -10 分 |
| 6 | 不遵守承诺卡片 | Card | 第 3 项 | 🟢 绿色标识，"不遵守承诺" -3 分 |
| 7 | 自定义纠正 | Card | 第 4 项 | ➕ 图标 + 输入框，自定义行为描述和扣分数 |
| 8 | 选中汇总 | SummaryBar | 底部上方 | "已选择 N 项 共 -XX 分" |
| 9 | 确认纠正按钮 | Button | 最底部 | 主色按钮 |

## 交互说明

- 卡片点击 → 选中/取消选中（多选）
- 选中态：卡片边框高亮 + 勾选标记
- 自定义纠正 → 输入行为描述 + 设置扣分数值
- 汇总实时更新已选数量和总扣分
- 确认纠正 → 提交 → 扣减积分 → 结果页
- 每个行为可点击查看详情/历史

## 配色方案

- 严重违规 (说谎): 红色色条 (#E53E3E)
- 中度违规 (打人): 黄色/橙色色条 (#DD6B20)
- 轻度违规 (承诺): 绿色色条 (#38A169) / 表示轻微
- 卡片: 白色背景 + 左侧色条
- 选中态: 蓝色边框 (#0077B6)
- 扣分数字: 红色 (#E53E3E)
- 汇总: 浅灰背景

## 海洋主题元素

- 卡片使用圆角气泡风格
- 严重程度色条使用波浪形边缘
- 背景有淡蓝纹理

## 优化建议

1. 增加严重程度分级（红/橙/黄/绿）
2. 增加纠正频次限制（同行为不可重复纠正）
3. 增加"取消纠正"功能（家长可撤回）
4. 增加纠正后教育建议（如：说谎后应引导孩子诚实）
5. 增加纠正次数统计面板
6. 自定义纠正可保存为常用模板
