```

│  │   小明     ▼          │    │  (下拉切换孩子)
│  └──────────────────────┘    │
│                               │

│  │    ⭐ 150             │    │  ← 积分大数字展示
```

## 组件清单

| # | 组件 | 类型 | 位置 | 说明 |
|---|------|------|------|------|
| 1 | 导航栏 | AppBar | 顶部 | 左侧返回按钮 ←、标题"积分中心" |
| 2 | 孩子选择器 | Dropdown | 导航栏下方 | 当前孩子名 + 下拉箭头 ▼，点击切换孩子 |
| 3 | 积分总览卡片 | Card | 选择器下方 | 大号 ⭐ 积分数字(150)、标签"当前积分" |
| 4 | 今日统计 | StatBadge | 积分卡片内 | 收入(+50) / 支出(-20)，不同颜色区分 |
| 5 | 积分流水列表 | List | 积分卡片下方 | 每项：日期 + 行为描述 + 分数变化，正分绿色/负分红色 |
| 6 | 查看更多 | Link | 列表底部 | "查看更多 →"，点击跳转完整积分历史 |
| 7 | 底部 TabBar | BottomNav | 底部固定 | 同首页一致 |

## 交互说明

- ← 返回 → 回到首页
- 孩子选择器 ▼ → 弹出下拉列表，选择不同孩子查看对应积分
- 列表项点击 → 查看该条积分详情
- "查看更多 →" → 跳转积分历史完整页面
- 底部 TabBar 切换页面

## 配色方案

- 导航栏: 海洋蓝 (#0077B6)
- 积分卡片: 白色 + 蓝色渐变边框
- 积分数字: 金色 (#FFD700) ⭐ + 大号字体
- 正分: 绿色 (#38A169)
- 负分: 红色 (#E53E3E)
- 列表: 白底，交替浅灰行
- 底部 TabBar: 白色背景，选中蓝色

## 海洋主题元素

- 积分卡片带有波浪形边缘装饰
- 背景有微妙的水波纹理
- 积分数字用气泡风格圆圈包围
- 下拉选择器使用海洋风格

## 优化建议

1. 积分数字增加计数动画（从旧值滚动到新值）
2. 今日收支增加环比趋势箭头（↑↓）
3. 积分流水可按周/月切换筛选
4. 增加积分日历热力图视图
5. 孩子切换时增加平滑切换动画
