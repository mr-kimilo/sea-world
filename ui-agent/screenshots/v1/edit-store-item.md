## 组件清单

| # | 组件 | 类型 | 位置 | 说明 |
|---|------|------|------|------|
| 1 | 导航栏 | AppBar | 顶部 | 返回 ←、标题"编辑商品" / "新增商品" |
| 2 | 商品名称输入框 | TextInput | 表单第 1 行 | 标签"商品名称:" + 文本输入框 |
| 3 | 图标选择器 | IconPicker | 表单第 2 行 | 标签"选择图标:" + 可横向滚动的图标网格 |
| 4 | 图标选项 | IconOption | 网格内 | 圆形/方形 Emoji 按钮，选中高亮 |
| 5 | 积分输入框 | NumberInput | 表单第 3 行 | 标签"所需积分:" + 数字输入框 |
| 6 | 库存输入框 | NumberInput | 表单第 4 行 | 标签"库存数量:" + 数字输入框(可选) |
| 7 | 保存按钮 | Button (Primary) | 表单底部 | 主色全宽按钮，文字"上架 / 保存商品" |
| 8 | 删除按钮 | Button (Danger) | 保存下方 | 红色文字按钮，仅编辑模式显示 |

## 交互说明

- 图标选择器滚动浏览 → 点击选中 → 高亮标记
- 积分数值可手动输入或步进器调整
- 保存 → 表单校验 → API 提交 → 成功返回商城列表
- 删除 → 确认弹窗 → 确认删除 → 返回商城
- 表单校验规则：名称必填、积分 > 0

## 配色方案

- 表单标签: 深灰 (#333)
- 输入框: 白色背景 + 浅灰边框 + 圆角
- 输入框焦点: 蓝色边框 (#0077B6)
- 图标选择: 选中态蓝色边框 + 浅蓝背景
- 保存按钮: 海洋蓝渐变
- 删除按钮: 红色 (#E53E3E)

## 海洋主题元素

- 图标选择器使用气泡式圆形按钮
- 输入框边框圆角较大（气泡风格）
- 保存按钮使用海洋渐变
