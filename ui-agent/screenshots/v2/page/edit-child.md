## V2 设计要点

- **新增趋势**：显示本周积分变化趋势
- **成就徽章**：显示已获得的徽章墙
- **积分报告链接**：快速跳转详细报告
- 与 add-child 风格统一

## V2 Prompt

```markdown
Design the "Edit Child" page for 海底世界 app — same form style as Add Child but pre-filled.

Layout:
1. Avatar: Current child photo with "点击更换头像"
2. Pre-filled form fields: 昵称(小明), 真实姓名, 出生日期, 性别(selected)
3. Read-only stats section:
   - ⭐ 当前积分: 150 (gold text, large)
   - 📈 本周趋势: +35 分 (green, with mini arrow chart)
   - 🏆 成就徽章 (3/8): display of earned badge emojis [🔥] [🌟] [💪] with empty slots shown as outlines
4. Save button: "🌊 保存修改"
5. Delete: "🗑️ 删除孩子" red button with double-confirmation dialog
6. Link: "📊 查看完整积分报告 →"

Style: Same ocean theme as add-child. Stats section with frosted glass card background. Badge wall with earned/unlocked visual states.
```
